## 技术实现

### 核心特性
- **智能搜索源选择**：根据查询类型自动选择最佳搜索源
  - 代码查询 → GitHub → Stack Overflow
  - 新闻查询 → Hacker News → Reddit
  - 通用查询 → DuckDuckGo → Reddit

- **成功率追踪**：每个搜索源独立追踪成功/失败率
- **自动故障切换**：失败时自动尝试下一个搜索源
- **结果去重**：自动去除重复 URL

### 关键突破

**Reddit API 修复**：通过添加浏览器头部，成功绕过反爬虫检测，成功率从 0% 提升到 100%
